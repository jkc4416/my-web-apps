# Deployment Guide

## Platform

- **Hosting:** Vercel
- **Domain:** www.funappbox.com
- **SSL:** Automatic via Vercel
- **Redirects:** non-www → www (configured in Vercel dashboard)

## Auto Deployment

Every push to `main` triggers an automatic Vercel deployment.

```bash
git add .
git commit -m "your message"
git push
```

Typical deploy time: 1~2 minutes.

## Environment

No environment variables required. All configuration is in source code:

- GA4 ID: hardcoded in `src/app/layout.jsx`
- AdSense ID: hardcoded in `src/app/layout.jsx`
- Verification codes: hardcoded in `src/app/layout.jsx`
- Site URL: hardcoded in `src/app/sitemap.js` and `src/app/layout.jsx`
- ads.txt: `public/ads.txt`

## Domain Setup (Vercel)

1. Vercel Dashboard > Project Settings > Domains
2. Primary domain: `www.funappbox.com`
3. Redirect: `funappbox.com` → `www.funappbox.com` (307)

### SEO Features

- JSON-LD `WebSite` structured data on hub page
- Per-page metadata with Korean long-tail keywords
- `title.template` (`%s | FunAppBox`) for consistent branding
- Priority-differentiated sitemap (hub 1.0, popular 0.9, tools 0.8, games 0.7)
- **Dynamic sitemap lastModified** — uses `new Date()` per build (re-crawl friendly)
- Fortune app set to daily changeFrequency
- `viewport` export with themeColor `#0a0a0f` for mobile browser UI

## Code Quality Audits

Multiple rounds of systematic review covered:
- localStorage try/catch for Safari private mode
- Clipboard/share API error handling
- Timer/interval leak prevention (ref-based cleanup)
- requestAnimationFrame cleanup balance
- Input validation (NaN/range/date round-trip)
- Mobile responsive canvas
- Event listener cleanup
- Accessibility (aria-labels)
- No console.log or debugger statements
- React keys (deterministic, no Math.random)
- Race condition fixes (functional setState for high scores)
- Memo optimization for expensive renders
- Dynamic sitemap date for SEO
- Korean IME composition handling
- theme-color meta for mobile
